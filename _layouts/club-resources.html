Club Resources

We have provided some apps for our memebers to use in their InfoSec Journey. Some of these applications require authentication. The file servers (public and private) require SFTP to upload files and download. Only a web browser is required to download files.

We plan to add more services in the future. If you have any question, want to report an issue, or require authentication please DM Scott Lagler(SmoothHacker) on either Slack or Discord.

  • Minecraft Server at mc.ufsit.club
  • Private Git Server
  • Private File Server
  • UFSIT's Juiceshop Instance
  • Ghidra Server for CTF collaboration
  • Public File Server
